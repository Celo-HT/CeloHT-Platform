# CeloHT,Open Source Financial & Environmental Platform on Celo

CeloHT is an open-source platform that enables financial inclusion, education, agent operations, and reforestation using **Celo**, **cUSD**, **CELO**, **Valora**, **IPFS**, and **The Graph**.

## Core Pillars
- Education . Blockchain & financial literacy
- Agent Operations . Cash-in/cash-out services
- Reforestation . Tree planting & environmental verification

## Features
- Fast mobile payments  
- Low fees  
- Agent network  
- Community governance  
- Open-source contributions

## Installation
```bash
git clone https://github.com/Celo-HT/CeloHT.git
cd CeloHT
npm install
npx hardhat compile
